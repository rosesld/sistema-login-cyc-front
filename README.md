# Frontend - Sistema Login Prueba Técnica C&C

Este proyecto es el frontend de la aplicación web de gestión de usuarios con autenticación y control de acceso por roles. Está desarrollado con HTML, CSS, JavaScript y jQuery.

## Características

- Interfaz sencilla y funcional.

- Login con validación básica.

- Visualización de datos según rol (admin, supervisor, usuario).

- Logout funcional.

- Uso de sesiones almacenadas en localStorage.

- Integración con backend para autenticación y datos.


## Despliegue

La aplicación está desplegada en Netlify y se puede acceder en:

[https://sistema-login-cyc.netlify.app](https://sistema-login-cyc.netlify.app)

## Instrucciones para uso local

1. Clona el repositorio

```
git clone https://github.com/rosesld/frontend-login-cyc.git
cd frontend-login-cyc
```

Abre `index.html` en tu navegador

(No requiere servidor local para funciones básicas)

## Notas

- Este frontend consume un backend desarrollado en Python y Litestar.

- Puedes modificar el archivo config.js para ajustar la URL del backend si es necesario.
